🔢 List & Key (10장, p.297~)

List

순서(index) 있음

중복 데이터 허용

예: ArrayList, LinkedList

for문 사용 가능 (순서가 있기 때문)

Set

순서 없음

중복 데이터 불가

예: HashSet, TreeSet

Iterator로 데이터 접근

⚡ JavaScript Arrow Function

표준형: (파라미터) ⇒ { return ~ }

파라미터가 1개일 때: 파라미터 ⇒ { return ~ } (괄호 생략 가능)

return 한 줄만 있을 때: (파라미터) ⇒ 결과값 ({}와 return 생략 가능)
